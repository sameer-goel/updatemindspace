## Features

- **Superpower Activation**: Interactive experience with video sequences
- **6-Question Journey**: Progressive superpower activation questions
- **Book Experience**: Interactive book reader with navigation
- **Responsive Design**: Works on all device sizes
- **Self-contained**: All CSS and JavaScript inline

## Usage

1. Start with `pages/index.html`
2. Follow the superpower activation journey
3. Complete the 6 questions
4. Explore the book content
